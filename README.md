# Advanced-Mathematics-and-Programming-for-Machine-Learning-and-Data-Science

course is divided into the following modules.

Advanced Mathematics
Probability distributions (discrete & continuous), vectors and matrices, linear transformations, eigenvalues, partial derivatives, and gradients.
Database Design & Data Modelling
Core relational model concepts, normalisation, and basic relational and non-relational data modelling approaches.
Advanced SQL
SQL fundamentals (DDL, DML, joins, filters, grouping), structured problem solving, window functions, CASE, partitions, indexing, and query optimisation.
NoSQL & MongoDB
NoSQL principles, CAP theorem, CRUD operations, MongoDB querying, and aggregation pipelines.
Advanced Python Programming
Object-oriented programming, problem-solving strategies, advanced Python programming concepts, and prompt engineering.
Python Data Science Libraries
Data manipulation and visualisation using NumPy, Pandas, Matplotlib, Seaborn, and an introduction to interactive tools like Plotly and Bokeh.
